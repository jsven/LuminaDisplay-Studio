# 7-inch Amazon 3D Scene Pack

## 定位摘要

- 目标客群：树莓派/开发板用户、工控与控制面板用户、车载/便携项目玩家、需要小尺寸触摸屏的 DIY 和嵌入式客户。
- 核心卖点：7 英寸 16:10 IPS、800x1280、全贴合触摸、400 nits、9mm CNC 铝合金机身、249g、Mini HDMI + 双 Type-C、双扬声器、75 x 75 mm VESA。
- 视觉关键词：小尺寸但不廉价、触摸灵敏、金属一体感、嵌入式友好、适合便携和开发套件。

## Amazon 出图规则

- 主图只放产品本体，纯白背景，不能加文案、图标、角标或不随货配件。
- 7 寸产品本身体积小，主图镜头要更紧，让产品占画面 85% 以上。
- 副图优先突出触摸、紧凑体积、开发板/控制面板兼容和接口完整性。

## 渲染统一规则

- 镜头比 16 寸版本更紧，避免 7 寸产品在画面里显得过小。
- UI 和图标尺寸要放大，确保小尺寸屏幕仍能被一眼读懂。
- 触摸交互只做轻微波纹或按钮高亮，不做夸张特效。
- 材质依然从 `metalness 0.8 / roughness 0.3` 起调，但边缘高光要更锋利，以强调 CNC 加工感。
- 所有图片统一突出“触摸 + 紧凑 + 嵌入式兼容”三条主线，避免照搬 16 寸的大屏办公叙事。
